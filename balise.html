Liste des balises

Voici une liste des balises avec une courte définition

Attributs Définitions
span Elément en ligne permettant d'insérer du texte (souvent utilisée pour donner un style différent au texte se trouvant dedans)
div Permet de créer un groupement de blocs pour aider à la disposition des éléments ayant un même type d'informations/style
p Créer un paragraphe
a Permet de créer des liens entre les pages et le web
h1-h6 Permet de définir un titre
br Permet de passer à la ligne (auto fermante)
hr Permet de tracer une ligne horizontale entre les éléments (auto fermante)
Strong Permet de marquer l'importance d'un texte
em Permet de mettre en évidence une partie d'un texte
i Permet de mettre en italique sans CSS
del Permet de supprimer visuellement une partie d'un texte
sup Permet de mettre un texte en exposant
pre Permet de créer un bloc pré formaté pour l'affichage de code source en brut
time Permet de définir une heure ou une date
dfn Permet de définir le mot auquel la définition actuelle fait référence
blockquote Permet d'indiquer que son contenu fait référence à une citation
footer Fin de la balise blockquote pour indiquer l'auteur de la citation
attribut cite Qui se met dans le footer ou le blockquote pour indiquer l'url
q Pour une citation courte
img Pour afficher une image
figure Permet de définir une figure
figcaption Permet de mettre un nom à une image
progress Permet de mettre une barre de progression
audio Permet d'ajouter un audio
vidéo Permet d'ajouter une vidéo
Table Créé un tableau
thead Entête de tableau
tbody Corps du tableau
Tr Ligne du tableau
Td Cellule du tableau
th Titre de cellule
tfoot Pied du tableau
