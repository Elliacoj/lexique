Liste des élements Css

Voici une liste des élements Css avec une courte définition

Attributs Définitions
background-attachment Vous permet de spécifier si l’image est fixe ou si elle doit défiler avec le contenu.
background-color Changer la couleur de font.
background-image Vous permet de mettre une image de fond à un élément ou à toute la page.
background-repeat Spécifie la manière dont une image de fond doit être répétée ou non.
background-size Spécifie la manière la taille de l’image de fond.
border Créé une bordure.
border-collapse Permet de spécifier si les bordures d’un tableau sont séparées ou si elles sont collées.
border-radius Vous permet de créer des bords arrondis.
border-style Vous permet de spécifier le style de la bordure.
box-shadow Vous permet de créer une ombre portée sur une boîte.
caption-side Permet de spécifier la position du caption du tableau ( légende, si présente ).
clear Replace le flux de la page en dessous des éléments flottants à gauche, à droite, ou les deux.
color Change la couleur du texte.
cursor Vous permet de changer le type de puces d’une liste, ordonnée ou non.
display La propriété display va permettre à des éléments ayant habituellement un comportement en ligne ou en bloc ( ou tableaux ) de se comporter comme un autre type!
empty-cells Permet de spécifier le comportement à adopter face à des cellules vides.
float Retire l’élément du flux normal pour le placer à gauche, à droite, ... dans et par rapport à son conteneur.
font-family Change la police de caractère.
font-size Change la taille de la police.
font-style Permet de mettre un effet au texte (italique).
font-variant Permet de choisir entre de petites lettres capitales et un texte normal.
font-weight Permet de mettre un texte en évidence (gras).
height Change la hauteur.
line-height Change la hauteur de ligne.
list-style-image Vous permet d’utiliser une image à la place des puces.
list-style-type Vous permet de changer le type de puces d’une liste, ordonnée ou non.
list-style-position Vous permet de définir le retrait des éléments dans la boîte ( le retrait ).
margin (top, bottom, left, right) Permet de spécifier une marge à l'extérieur.
opacity Permet de gérer la transparence.
overflow Vous permet de gérer les débordements de contenu.
padding (top, bottom, left, right) Permet de donner une marge intérieure.
position Permet de définir la manière dont vos éléments sont placés dans le flux.
text-align Permet d'aligner un texte.
text-decoration-color Vous permet de définir la couleur de la décoration du texte.
text-decoration-line Permet de choisir le type de ligne de la décoration.
text-decoration-style Vous permet de définir le style de la ligne de la décoration.
text-indent Permet d'indenter la permière ligne de paragraphe.
text-shadow Permet de mettre une ombre à un texte.
text-transform Permet de choisir entre majuscules et minuscules.
vertical-align Permet d’aligner un texte de balise en ligne par rapport à son paragraphe.
visibility Vous permet de spécifier si un élément est visible ou non
width Permet de gérer la largeur.
word-wrap Défini le comportement de césure si un long mot doit être coupé et passé à la ligne.
z-index Vous permet de placer des éléments par dessus d’autres éléments dans le cas ou ils venaient à se chevaucher.
