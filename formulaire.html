Liste des balises pour formulaire

Voici une liste des balises pour formulaire avec une courte définition

Attributs Définitions
form Permet de spécifier que le bloc est un formulaire
fieldset Utilisé pour grouper des champs
label Utilisé pour donner une description à un champs
input Champs de type texte, ail, password, nom, submi
textarea Définir une zone de texte
select Définir une liste de choix
option Définir une valeur de "select"
optgroup Définir un groupe d’options
